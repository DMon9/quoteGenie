API Documentation

Welcome to the EstimateGenie API. Get started with intelligent construction estimates powered by AI.

Base URL

https://api.estimategenie.net/api

Quick Start

Get your first estimate in minutes. Here's a simple example: 

curl -X POST "https://api.estimategenie.net/api/v1/quotes" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -F "file=@project-photo.jpg" \
  -F "project_type=kitchen" \
  -F "description=Kitchen remodel with new cabinets"

Authentication

EstimateGenie uses API keys to authenticate requests. Get your API key from your dashboard.

Keep your API key secure

Never expose your API keys in client-side code or public repositories.

Include your API key 

Authorization: Bearer eg_sk_live_YOUR_API_KEY

API Endpoints

Create a Quote

POST /v1/quotes

Upload project images and details to generate an AI-powered construction estimate.

Request Parameters

Parameter Type Required Description
file file Yes Project image (JPG, PNG, max 10MB)
project_type string Yes Project category (kitchen, bathroom, etc.)
description string Yes Detailed project description
quote_type string No basic, detailed, or project-plan

List Quotes

GET /v1/quotes

Retrieve a list of all your quotes with optional filters.

Get Quote Details

GET /v1/quotes/{id}

Retrieve details of a specific quote by ID.

JavaScript SDK

Official JavaScript/Node.js SDK for seamless integration.

Installation 

# Install via npm
npm install @estimategenie/sdk

# Install via yarn
yarn add @estimategenie/sdk

Basic Usage 

import { EstimateGenie } from '@estimategenie/sdk';

// Initialize client
const client = new EstimateGenie('YOUR_API_KEY');

// Create a quote
const quote = await client.quotes.create({
  file: imageFile,
  project_type: 'kitchen',
  description: 'Complete kitchen remodel',
  quote_type: 'detailed'
});

console.log('Quote ID:', quote.id);
console.log('Total Cost:', quote.total_cost.amount);

Error Handling

EstimateGenie uses HTTP response codes to indicate success or failure.

Code Status Description
200 OK Request succeeded
400 Bad Request Invalid parameters
401 Unauthorized Invalid API key
429 Rate Limited Too many requests
500 Server Error Internal server error

Best Practices

Provide Detailed Descriptions

The more detail you provide, the more accurate the estimate. Include dimensions, materials, and special requirements.

Use High-Quality Images

Clear, well-lit photos from multiple angles provide better AI analysis and more accurate estimates.

Handle Errors Gracefully

Implement proper error handling with retry logic and exponential backoff for failed requests.

Cache Results

Store quote results locally when appropriate to reduce API calls and improve response times.
